Localserve
==========

Localserve creates a local HTTP server using the python HTTP module.
Mainly aimed at transfering files from your pc with python3 and if you run linux(debian)
I know very sepcific use case. But does what it says.

Commands to download the repo:
```
git clone https://github.com/Sadan0/about-me.git

```

To Just get the Executable Use:

download the file localserve file
(From the folder it exsist in, usually in ~/Downloads)
```
sudo mv localserve /usr/local/bin/

```
then 

```
sudo chmod +x /usr/local/bin/localserve

```
Now run it from any dir that you want to download files form.
Type the url given by the the program in a url and you will be able to download all files from that dir.
